# psarch.github.io

The Application utilizes Android’s Accessibility Service to observe call transcripts in real-time. This data is processed solely to navigate automated phone menus (IVR). We do not store, upload, or share any transcript data to external servers. All analysis performed by the integrated AI model is ephemeral and deleted immediately upon the termination of the active call session.

Users acknowledge that the Application may utilize third-party AI models (e.g., Google Gemini) to interpret call context. While data is sent to these models for processing, no PII (Personally Identifiable Information) is intentionally stored by the Application. However, the Application cannot control the data retention policies of third-party model providers.
